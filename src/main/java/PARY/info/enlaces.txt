////////////----------Actividad-------/////////////////
GET
____
Devuelve todas las Actividad
-> http://localhost:8080/api/actividades

Devuelve una Actividad en especifico
-> http://localhost:8080/api/actividades/buscarID/{id}

Devuelve todas las Actividades que contengan un nombre en especifico
-> http://localhost:8080/actividades/buscarNombres/{nombre}

Devuelve todas las Actividades ordenadas por fecha
-> http://localhost:8080/api/actividades/recent

Devuelve la cantidad de comentarios de la actividad
-> http://localhost:8080/api/actividades/comentario/cantComentarios/{id}

Devuelve la cantidad de me encanta de la actividad
-> http://localhost:8080/api/actividades/cantMeEncanta/{id}

Devuelve los comentarios de la actividad
-> http://localhost:8080/api/actividades/comentario/findAllComentarios/{id}

Devuelve la cantidad de Activides
-> http://localhost:8080/api/actividades/cantidad

POST
_____
Crea y devuelve una Actividad----------------RequestBody
-> http://localhost:8080/api/actividades/crearAct

PUT
____
Actualiza una Actividad en especifico----------------RequestBody
-> http://localhost:8080/api/actividades/actualizarAct/{id}

PATCH
______
Agrega un comentario del perfil----------------RequestBody
-> http://localhost:8080/api/actividades/comentario/AgregarComentario/{id}+{idP}

Actualiza los comentarios de una actividad----------------RequestBody
-> http://localhost:8080/api/actividades/comentario//CambiarComentario/{id}+{idP}

Annadir un me encanta de un perfil
id: id de la actividad, idP: id del perfil, addOrDel: valor del boton si esta activado o no
-> http://localhost:8080/api/actividades/addOrDelMeEncanta/{id}+{add}+{idP}

Agrega o elimina la actividad de MisActividades
-> http://localhost:8080/api/actividades/addActMisActiv/{id}+{add}+{idP}

Crea una reservacion si el param reser = true
-> http://localhost:8080/api/actividades/creaReser/{id}+{reserv}+{idP}

DELETE
________
Elimina una Actividad en especifico
-> http://localhost:8080/api/actividades/deleteAct/{id}

Elimina todas las Actividades que ya hayan transcurrido
-> http://localhost:8080/api/actividades/deleteOldAct

Elimina todas las Actividades
-> http://localhost:8080/api/actividades/deleteAllAct
